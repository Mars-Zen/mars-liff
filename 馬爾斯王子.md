# 馬爾斯王子 — 專案概覽

## 這是什麼產品？

**馬爾斯王子** 是一個以 LINE 為主要介面的占星與靈性服務平台。使用者透過 LINE 聊天或開啟 LIFF 頁面，與 AI 進行占星對話、接收每日推播服務、並參與群組互動功能。

**主要功能：**
- AI 占星對話（輸入任何問題，AI 結合本命盤給出個人化星盤建議）
- 每日自動推播（今日運勢、穿搭建議、餐點推薦、能量波形圖）
- 四元素能量系統（傳照片可增加元素能量，能量低會收到通知）
- 星獸系統（AI 依照星盤生成專屬星獸角色，可升級進化）
- 太陽回歸（生日當天自動推播年度星盤報告）
- 星際群組（建立群組、邀請朋友，含週排行榜、心情札記、測驗對戰、雙人合盤、歷史戰績）

**LINE 圖文選單（6格）：**

| 格子 | 功能 |
|------|------|
| 推薦好友 | 分享機器人邀請連結 |
| 開啟相機 | 直接開啟 LINE 相機（傳圖觸發元素能量分析）|
| 每日答題 | 開啟每日問答 LIFF 頁面 |
| 星際群組 | 開啟星際群組 LIFF 頁面 |
| 星獸圖鑑 | 開啟星獸圖鑑 LIFF 頁面 |
| 召喚星獸 | 觸發召喚星獸功能 |

## 技術架構

### 整體運作流程

```
使用者傳 LINE 訊息 / 開啟 LIFF 頁面
        ↓
LINE Messaging API → n8n Webhook（主入口工作流）
        ↓
n8n 依訊息類型路由至對應工作流
        ↓
工作流呼叫外部 API（Gemini AI、Python 占星服務、CWA 天氣等）
        ↓
n8n 讀寫內建資料表（SQLite）
        ↓
n8n 透過 LINE Reply / Push API 回應使用者
```

**LIFF 頁面（前端）** 是靜態 HTML 檔案，透過 LINE 的 LIFF 框架在 LINE 內開啟，直接以 HTTP POST 呼叫 n8n Webhook。

### 技術棧

| 層級 | 技術 |
|------|------|
| 自動化引擎 | n8n（GCP VM 自架，Docker 容器） |
| 通訊管道 | LINE Messaging API |
| 前端介面 | 靜態 HTML/CSS/JS，透過 LINE LIFF 開啟，托管於 GitHub Pages |
| 主要 AI 模型 | Google Gemini 2.5 Flash（透過 Gemini API 直接呼叫） |
| 資料庫 | n8n 內建資料表（DataTable，底層為 SQLite） |
| 檔案儲存 | Google Cloud Storage（bucket：`mars-zen-astro-dev`） |
| 占星計算 | 內部 Python API（由 `PY_ASTRO_API_KEY` 驗證） |
